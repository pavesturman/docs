# Troubleshooting

## Common Issues

### 1. SDK Initialization Failed

- Check if access token is correct
- Ensure user ID is provided
- Verify iOS version (12.0+ required)

### 2. Deep Links Not Working

- Verify URL scheme configuration
- Check URL scheme matches dashboard settings
- Ensure proper deep link handling implementation

### 3. Referral Tracking Issues

- Verify user authentication
- Check network connectivity
- Ensure proper event tracking implementation
